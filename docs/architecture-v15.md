# v15 分类别名与分页跳转

## 目标

修复历史资产因旧网址与现有数据源网址不一致而变成“未知分类”的问题，并优化资产看板分页体验。

## 变更

- 新增 `source_url_aliases` 表，用旧网址映射到数据源标头。
- 内置旧网址映射：
  - `https://shengshi888.cc/` -> `盛世`
  - `http://wansheng888.cc/` -> `万盛`
  - `https://nbcz.top/` -> `嘉盛`
- 启动时自动按“旧网址别名 + 关键词”修复历史资产 `source_id`。
- 修改数据源网址时会自动记住旧网址作为别名，历史资产继续归属原标头。
- 资产看板分页增加页码输入与跳转按钮。

## 说明

如果“万盛｜管城”的网址从 `https://boxin888.cc/` 改为 `https://biying888.cc/`：

- 已经绑定了 `source_id` 的旧资产仍然归到“万盛｜管城”。
- 修改数据源时，旧网址会写入 `source_url_aliases`，后续未绑定但符合“旧网址 + 关键词”的历史资产也会被归回对应标头。
